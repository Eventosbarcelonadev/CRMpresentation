# Preguntas para Eventos Barcelona

---

## 1. Las propuestas comerciales

- Los PowerPoints que envias hoy, **cuantos diferentes tienes?** (uno por tipo de show, por tipo de evento, o uno generico que adaptas?)
- **Que cambias de una propuesta a otra?** Solo precio y nombre del cliente, o tambien cambias fotos, textos, shows incluidos?
- **Tienes una tabla de precios base** o calculas cada presupuesto desde cero?
- **Como decides el margen?** Porcentaje fijo, depende del cliente, del tipo de evento?
- **Nos puedes compartir 2-3 propuestas reales** que hayas enviado? Para ver la estructura exacta

## 2. El proceso comercial

- Cuando llega un lead, **siempre propones un solo show o varias opciones?**
- **Hay negociacion de precio** normalmente o es precio cerrado?
- **Que te pareceria que las propuestas fueran paginas web** en vez de PDFs? El cliente recibe un link, ve fotos/videos del show, precio, y puede aceptar directamente

## 3. Herramientas y accesos

- **La web es WordPress?** O que plataforma usa?
- **Tu plan de Holded** es de pago? (la API solo funciona en planes de pago)
- **Tienes acceso al panel DNS** de eventosbarcelona.com?

## 4. Base de datos existente

- Los contactos de artistas, **estan en algun sitio** (Excel, Mailchimp, carpeta de emails)?
- **Cuantos contactos** tiene tu Mailchimp aproximadamente?

---

*Con estas respuestas podemos definir exactamente como montar el sistema.*
